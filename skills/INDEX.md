# Skills Index

This file indexes available global and workspace skills for Antigravity.

- [backlog-refinement/SKILL.md](backlog-refinement/SKILL.md): Refines backlog tasks and aligns local-first backlog items with Jira.
- [debug-mantra/SKILL.md](debug-mantra/SKILL.md): Verbatim recite of debug mantra and four-step debugging discipline.
- [management-talk/SKILL.md](management-talk/SKILL.md): Rewrites engineering content for VPs, directors, PMs, or release managers based on target channels.
- [post-mortem/SKILL.md](post-mortem/SKILL.md): Formulate a post-mortem document after resolving bugs.
- [project-audit-reviewer/SKILL.md](project-audit-reviewer/SKILL.md): Review project code and architecture against rules.
- [scrutinize/SKILL.md](scrutinize/SKILL.md): Independent peer review of plans, PRs, and changes.
- [system-design-rules/SKILL.md](system-design-rules/SKILL.md): System architecture design standard covering trade-off analysis, APIs, and diagrams.
- [task-estimation/SKILL.md](task-estimation/SKILL.md): Estimating task complexity, description, and requirements.
- [confluence-sync/SKILL.md](confluence-sync/SKILL.md): Synchronizes project documentation files (Architecture, Specs, ADRs, and API docs) to Confluence.
- [git-workflow/SKILL.md](git-workflow/SKILL.md): Standardized Git branching, Pull Requests, releases, and GitHub CLI (gh) best practices.
