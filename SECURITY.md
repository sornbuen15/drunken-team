# Security Policy

## Supported Versions

We actively support and apply security patches to the following versions of Drunken Team Inn:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, please do NOT report it via a public GitHub Issue. Instead, please report it privately:

1. Send an email to the repository owner (**sornbuen15@gmail.com**).
2. Include a detailed description of the vulnerability, steps to reproduce, and any potential exploits or proof-of-concepts.

We will acknowledge your report promptly and work on resolving the issue responsibly. Thank you for helping keep this project safe!
